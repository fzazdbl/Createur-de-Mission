# Createur de Mission

Base locale offline-first pour gérer des missions bug bounty, des notes contextuelles, des preuves et le suivi administratif.

## Démarrage
1. Installer Node.js dependencies.
2. Lancer `npm run dev`.

## Fonctions prévues
- Wizard mission depuis un lien
- Notes contextuelles dynamiques
- Conformité `warn/strict`
- Dashboard compta + exports
- Preuves + index + hash
- Planning quotidien
